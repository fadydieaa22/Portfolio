# Project Images Folder

Add your project screenshots here:

1. `mealsapp.jpg` - Screenshot of your MealsApp project
2. `angular-project.jpg` - Screenshot of your Angular application
3. `mern-project.jpg` - Screenshot of your MERN stack application

**Recommended specs:**
- Resolution: 1920x1080 pixels (16:9 aspect ratio)
- Format: JPG or PNG
- File size: Under 500KB (compress if needed)

The portfolio will show emoji fallbacks if images are not found.
